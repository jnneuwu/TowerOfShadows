# Tower of Shadows — Unity Project

## Quick Start (4 Steps)

### Step 1: Create New Unity 2D Project
1. Open Unity Hub → New Project
2. Select **2D (Built-in)** or **2D Core** template
3. Unity version: **2022 LTS or later** (or Unity 6)
4. Name it "TowerOfShadows" → Click Create

### Step 2: Import Files
1. Close Unity
2. Copy the entire **Assets** folder from this zip into your project, replacing the existing one
3. Also copy **ProjectSettings/TagManager.asset** to your project's ProjectSettings/ folder (replace existing)
4. Re-open the project in Unity
5. Wait for all scripts to compile — Console should show 0 errors

### Step 2: Auto-Build the Game
1. In the Unity menu bar, click: **Tower of Shadows → Auto Setup Game (One Click!)**
2. A window will open — click the big **"BUILD ENTIRE GAME"** button
3. Wait ~10 seconds. It will create all scenes, sprites, prefabs, and set build settings.
4. A dialog will confirm "Done!"

### Step 3: Play!
1. Open the scene: **Assets/Scenes/MainMenu.unity**
2. Press **Play** in the Unity Editor
3. Click "START GAME" to begin

## Important Notes

### Before Running Auto Setup:
- Make sure there are **NO script errors** in the Console
- If you see errors about missing tags, add "Player" and "Wall" tags in Edit → Project Settings → Tags and Layers
- If button callbacks don't work, manually wire them:
  - MainMenu: StartBtn → MainMenuManager.OnStartGame()
  - MainMenu: QuitBtn → MainMenuManager.OnQuit()
  - In-game: Pause/Death buttons → UIManager methods

### Game Structure
- **MainMenu** → Start the game
- **Floor 1** — Shadow Walkers (purple, patrol AI)
- **Floor 2** — + Shadow Chargers (red, charge AI)
- **Floor 3** — + Shadow Casters (blue, ranged AI) + Final Boss
- **Victory** — You win!
- **Death** → Restart from Floor 1

### Entity Colors (Placeholder Shapes)
- 🔵 Cyan = Player (Kael)

### Customization
After auto-setup, you can:
- Edit tilemap rooms in each Floor scene
- Adjust enemy stats in the Inspector
- Add more enemies/crates to rooms
- Change colors by editing sprite PNGs in Assets/Sprites/
- Add sound effects to the AudioManager component
